# 超星学习通自动答题工具

基于 DeepSeek API 的自动答题脚本，支持选择题和判断题。

## 使用方法

### 1. 下载谷歌浏览器

下载地址：https://www.google.cn/chrome/index.html

下载完成后直接进入，选择不登录使用即可。

### 2. 获取 DeepSeek API Key

进入 [DeepSeek 官网](https://platform.deepseek.com/)，点击 API 开放平台，注册登录后充值（十块钱足够用很久），创建一个 API Key。

> ⚠️ **重要**：API Key 只显示一次，请立即保存到本地，不要发给别人！

### 3. 启动脚本

双击 `run.bat`，等待自动安装环境。

- 第一次启动会创建虚拟环境并安装依赖
- 按提示输入刚才创建的 API Key
- 然后回车

### 4. 获取作业网址

用另外的浏览器打开学习通网页版，复制作业页面的网址。

### 5. 运行

将复制的网址粘贴到终端中提示的 URL 处，然后回车。

脚本会弹出浏览器让你登录学习通，登录后看到作业页面，回到终端按回车（按提示操作即可）。

> ⚠️ **退出终端前**，一定要记得先暂时保存答案或者提交，不然退出终端后就没了。

## 工作原理

- **DeepSeek API** 负责识别题目并给出答案
- **Playwright** 负责模拟浏览器点击操作
- 目前支持：**选择题**、**判断题**

## 费用与准确率

- 💰 一百道选择题花费不到一分钱，开销可以当电费算
- ✅ 正确率在 95% 以上
- ⚠️ 极低概率会发生错很多的情况，不放心可以用豆包拍一拍检查几个
- 🔄 如果发现错很多，重新让脚本做一遍即可
- 🛡️ 若出现学习通提示"作业答题情况异常"，退出脚本再重新启动几次就好
